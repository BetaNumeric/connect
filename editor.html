Connect Level Editor

Tips / Controls

Add objects, drag them on the canvas, then export a `buildLevel` case snippet. Keyboard: Arrow keys move selected object, `Delete` removes it, `Ctrl+D` duplicates. Drag side/corner handles to resize, hold `Ctrl` for symmetric resize, hold `Shift` for uniform scaling. Drag the round handle above an object to rotate (hold `Shift` to snap). Use `Shift`/`Ctrl` click on canvas to multi-select, then `Combine To Group` or `Ungroup` (also detaches rotor parts). Use `Add To Rotor` to attach selected Box/Arc Box/Shape/Rigid Group objects to one selected rotor. Use `Arc Box` to create inward or outward circular sides on one or more sides.
